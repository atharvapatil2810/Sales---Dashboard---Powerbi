# 📊 Sales Performance Dashboard (Power BI)

An interactive data visualization project using the Sample Superstore dataset to track key business metrics including Total Sales, Profit, and Orders.

## 🚀 Key Features
* **Regional Analysis:** Insights into sales distribution across Central, East, South, and West regions.
* **Trend Analysis:** Line charts showing sales growth from 2015 to 2018.
* **Interactive Slicers:** Capability to filter data by specific regions instantly.

## 🛠️ Tools & Technologies
* **Power BI Desktop**
* **DAX** (Data Analysis Expressions)
* **Data Cleaning:** Handled within Power BI Query Editor
